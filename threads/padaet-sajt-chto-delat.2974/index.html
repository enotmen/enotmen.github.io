В любой непонятно ситуации, если не хотите разбираться в причинах - перезагружаете сервер reboot

Если хотите разобраться, то по этой инструкции cinemapress logs


Опять Nginx не запустился? Сколько Вы прописали фильмов на общих странице? Сервер не справляется.



Если ничего не помогает, делаете так:
  • Бэкап cinemapress backup убедитесь, что всё сохранилось;
  • Очищаете сервер командой cinemapress clear_vps всё удалит
  • Создаете сайт cinemapress install
  • Добавляете все Ваши доп. домены cinemapress domain
  • Восстанавливаете бэкап cinemapress backup
