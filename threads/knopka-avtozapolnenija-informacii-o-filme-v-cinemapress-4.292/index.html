Поставить кнопку автозаполнения


  1. Устанавливаете обезьянку:
    • для Google Chrome;
    • для Mozilla Firefox;
    • для Opera;
    • для Safari;
    • для Microsoft Edge;
  2. Нажимаете установку;
  3. Заходите в Админ-панель - Добавить фильм - Добавить/изменить фильм;
  4. Вверху нажать на кнопку Заполнить информацию, будет заполнена на основании ID КиноПоиск.


​

PS: Данный скрипт создан для пользователей, которые вручную добавляют фильмы на сайт.

PPS: Работает только через VPN, любой.
