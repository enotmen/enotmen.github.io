Я сам уже долгое время бьюсь с бедой. Но как были битые постеры так и остаются. Только исправлю в ручную появляются новые .

У меня прописанно вот так

# Заполнение ID TMDb + постер + красивое фото для фильмов, у которых уже есть ID IMDb

23 ~ db ~ custom.imdb_id ~ https://api.themoviedb.org/3/find/t...ddf4635473cf4cf456576&external_source=imdb_id ~ movie_results.0.id <> custom.tmdb_id <> 1 ~ movie_results.0.poster_path <> poster <> 1 ~ movie_results.0.backdrop_path <> picture <> 1

# Заполнение ID TMDb + постер + красивое фото для сериалов, у которых уже есть ID IMDb

23 ~ db ~ custom.imdb_id ~ https://api.themoviedb.org/3/find/t...ddf4635473cf4cf456576&external_source=imdb_id ~ tv_results.0.id <> custom.tmdb_id <> 1 ~ tv_results.0.poster_path <> poster <> 1 ~ tv_results.0.backdrop_path <> pictures <> 1

# Заполнение рейтинга КиноПоиск и IMDb, напрямую с КиноПоиска.

23 ~ db ~ kp_id ~ https://rating.kinopoisk.ru/[id].xml ~ rating.kp_rating._attributes.num_vote <> kp_vote ~ rating.kp_rating._text <> kp_rating ~ rating.imdb_rating._attributes.num_vote <> imdb_vote ~ rating.imdb_rating._text <> imdb_rating

# Заполнение постера с Bazon (постер с КиноПоиск, Яндекс, Shikimori, TMDb, IMDb)

23 ~ db ~ ~ https://bazon.cc/api/search?token=2848f79ca09d4bbbf419bcdb464b4d11&kp=[id] ~ results.0.kinopoisk_id <> kp_id <> 1 ~ results.0.info.poster <> poster <> 1

# Заполнение постера с Kodik (постер с Shikimori)

23 ~ db ~ ~ https://kodikapi.com/search?token=b...435&with_material_data=true&kinopoisk_id=[id] ~ results.0.kinopoisk_id <> kp_id <> 1 ~ results.0.material_data.poster_url <> poster <> 1
