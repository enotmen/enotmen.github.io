Данный параметр появляется из-за настройки в CloudFlare - SSL Flexible, сайт видит что к нему подключились по HTTP, потому перенаправляет на HTTPS.

Посетитель -> CloudFlare HTTPS -> VPS HTTP -> Сайт



Сгенерировав собственный сертификат командой cinemapress https и выбрав настройку в CloudFlare - SSL Full (strict), префикс добавляться не будет, так как сайт будет видеть HTTPS протокол.

Посетитель -> CloudFlare HTTPS -> VPS HTTPS -> Сайт
