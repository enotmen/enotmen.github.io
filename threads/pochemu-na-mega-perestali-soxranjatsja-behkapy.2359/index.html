Проверьте соединение Сервера и Mega командами:
  • DOMAIN="example.com"
  • DOMAIN_=$(echo "${DOMAIN}" | sed -r "s/[^A-Za-z0-9]/_/g")
  • docker exec "${DOMAIN_}" rclone mkdir CINEMAPRESS:/check-connection
Должна создаться папка check-connection на Mega.

Используйте пароль для Mega только из цифр и английских букв.

  • Создание конфигурации: cinemapress backup example.com config "EMAIL" "ПАРОЛЬ"
  • Создание бэкапа: cinemapress backup example.com create
  • Создание конфигурации: cinemapress static example.com "EMAIL" "ПАРОЛЬ"
  • Создание бэкапа: cinemapress static example.com create

Можете перед этим удалить старые конфиги:
rm /home/example.com/config/production/rclone.conf
rm /var/rclone.conf

Создать пустой конфиг:
touch /home/example.com/config/production/rclone.conf
