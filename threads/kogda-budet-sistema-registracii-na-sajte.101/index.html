Нажмите для раскрытия...
Сейчас система полностью изолирована от внешнего вмешательства, т.е. работает исключительно на вывод информации, а ввод только через админ-панель. Это исключает множество атак на сайт. Под регистрацию структура подготовалена, но вводить её пока не планируется.

По поводу копирайтеров, то писать по одному описанию не вариант, ввиду большого числа фильмов. Заказываете у копирайтера сразу сотни описаний, чтобы они делали в таком формате (заносите в файл /config/update/default.json):
JSON: 
{
  "movies": [
    {
      "kp_id": ID_КиноПоиск,
      "description": "Уникальное описание"
    },
    {
      "kp_id": 298,
      "description": "..."
    },
    {
      "kp_id": 299,
      "description": "..."
    }
  ]
}
Дальше просто отправляете их на сайт одной командой:

wget git.io/JGKNq -qO i && chmod +x i && ./i update

можно сделать псевдо регистрацию, повесить на уника куку и под нее сделать акк, и сессия будет висеть пока кука имеется. вопрос для чего , добавлять какие то видео в свой лист для потом посмотреть, или же надо от этого имени комменты еще писать? вопрос если необходимо вести аналитику по конкретному пользователю это уже другое дело и затраченные ресурсы, вопрос зачем нужно человеку который хочет кино делать акк, для гостя это ненужное дествие. надо учитывать что придется одевать шаблоны профайла пользователя, список профайлов, возожность переписки между пользователями, системы голосований, учитывайте что данный функционал будет иметь смысл на большом кол-ве трафика, пиратские сайты хлопают один за одним , а для того чтобы обрасти трафиком под вышеописанные задачи нужно время не один год, и вложения на траф приличные без этого се трафа не видать норм обьемов. я когда то делал тубы тоже с регистрацией профайлами и трафа было не одна сотня к , но все равно пользователи были не столь активны, они приходят за тем чтобы посмотреть и закрыть сайт, поэтому мне кажется надо сконцентрировать усилия на том чтобы дать унику то что он ищет , если не знает что ищет - предложить лучшее для него (удобная выборка конта и поиск ) и дать ему с небольшим количеством не раздражающей рекламы посмотреть конт и при этом заработать себе , и главное сделать так чтобы ему понравилось и он пришел еще раз )
