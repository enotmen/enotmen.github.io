В API для главной используйте только одну строку.

В качестве теста того, что всё работает, испольуйте API pleer.video:

https://pleer.video/episodes.json ~ results.0.kp_id ~ results.0.season ~ results.0.episode



В этом API все сериалы у Вас точно есть на сайте, а значит будут показаны.

Запустите крон вручную по SSH: cinemapress cron
