Если у Вас довольно мощный сервер и Вы не хотите, чтобы простаивали его ресурсы, можете использовать его для любой другой CMS, либо для связки PHP+MySQL на которую загрузите Ваш php-код.

Установка PHP+MySQL: cinemapress cms example.com php-mysql
Установка WordPress: cinemapress cms example.com wordpress
Установка Drupal: cinemapress cms example.com drupal
Установка Joomla: cinemapress cms example.com joomla
Установка DLE: cinemapress cms example.com dle

Создать бэкап всей базы MySQL: cinemapress cms example.com php-mysql backup
Файл бэкапа: /var/lib/cinemapress/dump/backup.sql
Восстановление всей базы MySQL: cinemapress cms example.com php-mysql restore
Файл восстановления: /var/lib/cinemapress/dump/restore.sql

По окончании, Вы получите данные от новой базы данных MySQL и панели ее администрирования Adminer.
