Сделать ссылку:
HTML: 
<a href="#">Популярные</a>
Под ней сделать невидимый список ссылок:
HTML: 
<span id="collections" style="display:none">
    <a href="/Коллекция1">Коллекция1</a>
    <a href="/Коллекция2">Коллекция2</a>
</span>
Навесить на кнопку событие показа этого списка:
HTML: 
<a href="#" onclick="document.getElementById('collections').style.display = 'inline-block'">Популярные</a>
