​
Собственный файловый менеджер теперь доступен на каждом сайте. Больше не потребуется для изменения логотипа, стилей или HTML-кода шаблона, скачивать и устаналивать сторонний софт (FileZilla, WinSCP, и т.п.).
Файловый менеджер работает на поддомене ftp, поэтому в DNS должна быть А-запись - ftp - IP сервера

После добавления поддомена ftp, активируйте в админ-панели Файловый менеджер.


  • Создание мастер-пароля для доступа к конфигурации
​

  • Игнорируете предупреждение о SSL, у Вас он будет от CloudFlare (или собственный)
​

  • Удаляете все существующие записи
​

  • Создаёте SFTP запись для доступа к серверу
​

  • Заполняете данными, которые получили от хостинга
​

  • Делаете удобным, управление файловым менеджером (название, автоподключение, показ SVG картинок)
