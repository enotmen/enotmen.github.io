Первая проблема в нехватке памяти.

Вам нужно удалить все неактивные контейнеры:
docker rm $(docker ps -a -q)

Удалить старый образ:
docker rmi -f cinemapress/app:latest

Если запущен почтовый сервер, то остановите его на момент создания:
Остановить: docker stop mail
Запустить: docker start mail

Вторая проблема будет исправлена ночью, завтра уже точно генерация будет работать, так как перед ней всё будет очищаться автоматически.

Завтра сгенерируете:
cinemapress upd && cinemapress app "hd-films720.club" linux
