вы можете показывать фильмы для определенных стран, для всех остальных будут трейлеры: https://cinema.press/admin/blocking.html#blocking5

если пришла жалоба на фильм, можете просто сделать так, Описания фильмам - Добавить описание - Вводите ID - Информационные данные фильма - Плеер, вводите туда плеер Moonwalk с блокировкой в России - http://moonwalk.cc/serial/9b2221ae7967aad9195750a5dccf2bb5/iframe?block_ru=1
фильм будет показан всем кроме пользователей из России.
