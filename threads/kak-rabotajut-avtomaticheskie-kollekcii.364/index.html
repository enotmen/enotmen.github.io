Автоматические коллекции работают так:
  1. Создается коллекция, например с URL-ключем filmy
  2. Этот ключ добавляется в автоматическое обновление, например с Moonwalk
  3. На Moonwalk добавляется новая серия сериала, например ТБВ
  4. Система смотрит, есть ли сериал ТБВ на сайте?

  • ДА - ID сериала добавляется вверх коллекции;
  • НЕТ - ничего не происходит.
То, что Вы добавляете новый сериал на сайт, не добавит его в коллекцию, пока новая серия на Moonwalk не появится.
