Команда cinemapress update, делает такие шаги:
  1. Создание нового бэкапа cinemapress backup
  2. Удаление сайта cinemapress remove
  3. Создание сайта cinemapress install
  4. Восстановление бэкапа cinemapress backup
Такой порядок должен быть, если сайт рабочий.

Если нерабочий то бэкап нерабочего сайта делать не нужно.

Восстанавливать из того бэкапа, который есть последний рабочий на Mega.
  1. Удаление сайта cinemapress remove
  2. Создание сайта cinemapress install
  3. Восстановление бэкапа cinemapress backup
