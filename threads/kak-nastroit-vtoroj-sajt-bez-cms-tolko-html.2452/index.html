Сделайте сайт PHP+MySQL с сертификатом:
cinemapress cms example.com php-mysql
cinemapress https example.com

Затем просто удалите в /home/example.com/config/production/nginx/conf.d/default.conf все location с обработкой PHP.
У Вас будет статический сайт.
