С ростом популярности сервиса, возникает загруженность с которой этот сервис не справляется. В абсолютном большинстве случаев ждать ответа от moonwalk приходится месяцами неделями и ответят они только на 3-4-5 посланное письмо. Так как видео-хостинг moonwalk сейчас является must-have любого онлайн кинотеатра, а владение токеном (чужим или своим) ничем не отличается, можете использовать любой из предложенных токенов для moonwalk:

6eb82f15e2d7c6cbb2fdcebd05a197a2
858facbc8b7d9061761e8540f46a2b3b
29b413aaa453a43fb4a91d81648aca5c

PS. Если и видео-хостинг HDGO не отвечает, используйте эти токены для HDGO:

aybpym3p0wi6keou7jbi5xqu
0ijs9yfywb4qbxfx0a7klwwl
88lx01lvhfioza84mifc15yn

PPS. При достижении 5 000 просмотров плеера, Вы можете получать доход с рекламы в плеере, поэтому получение своего токена для сайтов с посещаемостью - ОБЯЗАТЕЛЬНО.

Если сайт уже подключен - то по всем вопросам следует обращаться в поддержку через личный кабинет (в буднее время ответ последует максимум через несколько часов).
