я перешол на CloudFlare а сайт не работает на https удалил сгенерированый как написано тут
Если Вам потребуется отключить HTTPS, Вы должны будете открыть файл /etc/nginx/conf.d/ВашСайт.com.conf и удалить всё что находится между

# Non WWW start
...
# Non WWW start

# Only HTTPS start
...
# Only HTTPS end

# Enable HTTPS start
...
# Enable HTTPS end
Затем перезагрузить nginx service nginx restart.
