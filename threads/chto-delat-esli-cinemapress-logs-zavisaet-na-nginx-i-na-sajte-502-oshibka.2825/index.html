Сайт должен очнуться.​
​
2) Если в админ панели pagespeed включен (то необходимо выключить)​
​
3) В SSH нужно выполнить команды для создания бакапа и обновления CMS​
  • для обновления установочного пакета: cinemapress upd
  • для создания бэкапа - cinemapress backup (убедитесь, что успешно)
  • для обновления CMS - cinemapress update

Nginx занял почти всю отведенную ему память 973 МБ.
  • Обновите установочный: cinemapress upd
  • Создайте новый бэкап: cinemapress backup (убедитесь, что успешно)
  • Сделайте очистку VPS командой: cinemapress clear_vps
  • Создайте сайт: cinemapress install
  • Восстановите бэкап: cinemapress backup
  • После этого сделайте перезагрузку: reboot
Таким образом Вы будете точно знать, что от старой конфигурации Nginx ничего не осталось.
