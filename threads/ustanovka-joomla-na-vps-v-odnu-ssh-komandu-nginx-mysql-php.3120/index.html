Запускается Docker контейнер с MySQL (MariaDB 10) + PHP 7 (ionCube).
Произведены все настройки прав для успешной установки.

example.com - это домен сайта, заменить на свой.
/home/example.com - это папка сайта.
  1. Устанавливаете Docker: bash <(wget git.io/JGKNq -qO-)
  2. Устанавливаете Joomla: cinemapress cms example.com joomla
Код: 
------------------------------------------------------------------
----------------------------[ joomla ]----------------------------
------------------------------------------------------------------
Website: http://example.com
MYSQL HOST: mysql
MYSQL DATABASE: example_com
MYSQL USER: example_com
MYSQL PASSWORD: OTU3MzU1ZjM2
Adminer: http://example.com/cinemaadmin
USER: cinemaadmin
PASSWORD: ZmU1OGM3ZDFi
------------------------------------------------------------------
Website: http://example.com





Потребуется создать в корне сайта txt файл:
~# touch /home/example.com/installation/_JoomlajJKvydkLAN52IOQEimdAW.txt





Скопировать конфигурацию и занести в файл:
~# nano /home/example.com/configuration.php
Удалить папку с установкой:
~# rm -rf /home/example.com/installation



Adminer: http://example.com/cinemaadmin
