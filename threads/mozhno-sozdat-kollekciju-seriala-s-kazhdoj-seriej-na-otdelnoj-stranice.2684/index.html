Теперь уникальное название (title, t), описание (description, d) и H1 (h1, h) можно сделать для любой серии сериала (S01E06, s3e12, s05e03, S4E2).

Уникальное название, описание и H1 для 4 серии 1 сезона сериала Чужак.
  • s1e4.t
  • s1e4.d
  • s1e4.h
http://imgur.com/oTIewEc

Можно прописать сразу для всех серий в своём файле, и затем перенести в форму на сайте.

JSON: 
{
    "unique": true,
    "s1e4": {
        "t": "Уникальное название Чужака",
        "d": "Уникальное описание Чужака",
        "h": "Уникальный H1 Чужака"
    },
    "s1e5": {
        "t": "5 серия 1 сезона сериала Чужак в озвучке [translate]",
        "d": "5 серия 1 сезона сериала Чужак в озвучке [translate]",
        "h": "5 серия 1 сезона сериала Чужак в озвучке [translate]"
    }
}
JSON должен быть валидным (RFC 8259), проверять через jsonformatter.curiousconcept.com
