Удалить предыдущий конфиг и создать новый в верхнем регистре.

Команда, которая проверяет существование нужного конфига:
rclone config show 2>/dev/null | grep "CINEMAPRESS"

Посмотри для начала конфигурацию
rclone config show
я изначально делал с паролем конфигурации. Он не нужен, из за этого проблема. Попробуй полностью снести конфигурацию rclone,
rm /usr/bin/rclone
rm /usr/local/share/man/man1/rclone.1
И выполни настройку заново, только без пароля конфигурации
