  • Получаете основной плеер по API
  • Доп. плеер получаете через скрипт, для этого в шаблоне movie.ejs делаете кнопку
HTML: 
 <a href="javascript:void(0)" data-ahoy="dop-player">Доп. плеер</a>
<div id="dop-player" data-kinopoisk="<%- movie.kp_id %>"></div>
<script src="//yohoho.cc/yo.js"></script>
  • Чтобы активировалась кнопка трейлера, нужно добавить ссылку в Настройках URL
