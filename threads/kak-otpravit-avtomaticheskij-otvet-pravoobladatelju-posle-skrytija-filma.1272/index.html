есть несколько вопросов по функционалу "Запрещенные фильмы -> Автоматическое скрытие фильмов".

1. параметр "Белый список адресов электронной почты" нужен для того, чтобы проверять письма только из этого списка? при заполненном данном параметре будет проверять только эти адреса, а остальные игнорировать?

2. в кроне CP_cron.js вижу параметр hour === 1 - проверка идет каждый час или ровно в час ночи?

3. думали ли о том, чтобы после автоматического скрытия фильмов формировать автоматический ответ правообладателю?
например:
1. в модуле ставим новое поле "Автоматический ответ", в котором можно забить шаблон ответа
2. подключаемся к почте, видим новый запрос от правообладателя
3. скрываем указанные фильмы, сохраняя в массив replies адрес отправителя, идентификатор письма и тело письма для дальнейшего цитирования
4. формируем ответы правообладателям на основе сохраненного массива replies и шаблона "Автоматический ответ", сохраняя письма-ответы на почтовом сервере
5. опицонально ответные письма можно отправлять только на основе параметра "Белый список адресов электронной почты", то есть, если запрос не из этого списка, то ответы не отправляем
