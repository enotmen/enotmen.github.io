  1. Покупаете на Inferno бэкап-сервер
  2. Заходите по SSH на сервер сайта
  3. Заходите в контейнер (в домене вместо точек и дефисов - подчеркивание) docker exec -it домен_ru /bin/sh
  4. Конфигурируете rclone на Ваш FTP бэкап-сервер rclone config
  5. Имя (name>) должно быть CINEMAPRESS
  6. Выходите с контейнера exit
  7. Делаете бэкап cinemapress backup
