Обращаться к CSS в плеере и поменять через JS.

Загрузить свой стиль в iframe:
JavaScript: 
var cssLink = document.createElement("link");
cssLink.href = "loading-style.css";
cssLink.rel = "stylesheet";
cssLink.type = "text/css";
frames['yohoho'].document.head.appendChild(cssLink);
ID отвечающий за loading:
CSS: 
#yohoho-loading {
    z-index: 9999;
    position: absolute;
    left: 0;
    top: 0;
    width: 100%;
    height: 100%;
    background: #2A3440 url(data:image/svg+xml;base64,PHN2ZyB3aWR0aD0iMTIwIiBoZWlnaHQ9IjEyMCIgeG1sbnM9I…A1MCA1MCIgZHVyPSIxcyIgcmVwZWF0Q291bnQ9ImluZGVmaW5pdGUiLz48L3BhdGg+PC9zdmc+) 50% 50% no-repeat;
