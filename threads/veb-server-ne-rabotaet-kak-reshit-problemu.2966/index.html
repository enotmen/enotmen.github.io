На MEGA есть бэкап основного сайта?

Если бэкап есть, удалите оба сайта .club и .site:
  • cinemapress remove
После этого создайте нужный сайт:
  • cinemapress install
И восстановите бэкап:
  • cinemapress backup
Перезагрузите сервер:
  • reboot
