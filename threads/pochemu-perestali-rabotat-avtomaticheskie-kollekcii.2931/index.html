так?
Нажмите для раскрытия...
1. На https://www.kinopoisk.ru/popular/films/?quick_filters=films&tab=all (или на любой другой странице Кинопоиска) нажмите Ctrl + U
2. Далее Ctrl + A, а потом Ctrl + С
3. Переходите на https://kp.cinema.press/ и вставляете код в поле сочетанием клавиш Ctrl + V
4. Нажимаете кнопку "Получить все ID из HTML"
5. После нажатия, в поле вместо кода отобразятся ID, удаляете последние 3 (4871,412,416), а остальные копируете.
6. Заходите в админ-панель Вашего сайта, далее контент, выбираете нужную коллекцию, нажимаете "Редактировать список ID" и вставляете в поле (Ctrl + V), сохраняете.
7. Всё.
