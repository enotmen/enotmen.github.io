Теоретически да, можно отображать только фильмы с уникальным описанием.
Выполните команду:
cd /home/ВашСайт.net && sed -i "s/XMLPIPE2\": \"xmlpipe2/XMLPIPE2\": \"rt/g" process.json && pm2 restart process.json --update-env
Вернуть обратно:
cd /home/ВашСайт.net && sed -i "s/XMLPIPE2\": \"rt/XMLPIPE2\": \"xmlpipe2/g" process.json && pm2 restart process.json --update-env
