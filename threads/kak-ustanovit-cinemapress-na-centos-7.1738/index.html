Создаёте пароль сами командами по очереди:
Bash: 
DOMAIN="домен.com"
PASSWD="пароль"

OPENSSL=`echo "${PASSWD}" | openssl passwd -1 -stdin -salt CP`
echo "admin:${OPENSSL}" > "/home/${DOMAIN}/config/production/nginx/pass.d/${DOMAIN}.pass"
после этого
Bash: 
docker exec nginx nginx -s reload
