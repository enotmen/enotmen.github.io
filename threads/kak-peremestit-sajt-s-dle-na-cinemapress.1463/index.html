cinemapress upd && cinemapress import dle

Скрипт импорта загрузите на сайт с DLE в корень: https://raw.githubusercontent.com/CinemaPress/CinemaPress/master/files/dle2cinemapress.php
В нем замените example.com на Ваш домен на CinemaPress доменНаCinemaPress.com

Вначале идет команда создания импорта доменНаDle.com/dle2cinemapress.php?domain=доменНаCinemaPress.com
Далее происходит скачивание импорта доменНаDle.com/uploads/files/доменНаCinemaPress.com.xml

Возможные ошибки:
  • Могут потребоваться какие-то права дать для файла, например 0666
  • Если не получится соединиться с базой.
Раскомментируйте строчки:​
// require_once (ENGINE_DIR . '/classes/plugins.class.php');​
// require_once (DLEPlugins::Check(ENGINE_DIR.'/inc/include/functions.inc.php'));​
и закомментируйте​
include ENGINE_DIR.'/data/dbconfig.php';​
