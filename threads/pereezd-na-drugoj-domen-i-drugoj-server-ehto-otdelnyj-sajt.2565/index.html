Тоже столкнулся с подобной проблемой.

Решение такое:

1. На старом сервере в разделе фильмов добавляем такую строку 0 ~ database ~ ~ ~ kp_id <> kp_id ~ premiere <> premiere ~ type <> type ~ title_ru <> title_ru ~ title_en <> title_en ~ description <> description ~ pictures <> pictures ~ year <> year ~ country <> country ~ director <> director ~ genre <> genre ~ actor <> actor
2. Делаем бэкап cinemapress backup
3. Восстанавливаем на новом сервере бэкап cinemapress backup

После этих действий фильмы вместе с описаниями переносятся на новый сайт

Чтобы обновить рейтинг можно добавить 0 ~ database ~ kp_id ~ https://rating.kinopoisk.ru/[id].xml ~ rating.kp_rating._attributes.num_vote <> kp_vote ~ rating.kp_rating._text <> kp_rating ~ rating.imdb_rating._attributes.num_vote <> imdb_vote ~ rating.imdb_rating._text <> imdb_rating
