Сделайте замену файла /lib/CP_text.js на содержимое CP_text.js

Теперь в названиях будут работать такие ключи как фильмы, сериалы, мультфильмы и т.д, которые берутся из URL, если у Вас вместо фильмы стоит movies, то Вы должны использовать этот ключ (movies) {}



Код: 
(фильмы) { Фильм «[title]» }
(сериалы) { Сериал «[title]» }
(мультфильмы) { Мультфильм «[title]» }
(мультсериалы) { Мультсериал «[title]» }
(аниме) { Аниме «[title]» }
(передачи) { ТВ-передача «[title]» }


В новой версии этот CP_text.js будет включен.
