Вот этим способом подтвердить сайт не удалось:

В /config/production/nginx/conf.d/default.conf

Из этого
location ~* ^/(files|themes/.*/public|favicon.*|android-chrome.*|apple-touch-icon\.png|mstile-150x150\.png|browserconfig\.xml|safari-pinned-tab\.svg|site\.webmanifest) {

Сделать это
location ~* ^/(files|themes/.*/public|favicon.*|android-chrome.*|apple-touch-icon\.png|mstile-150x150\.png|browserconfig\.xml|safari-pinned-tab\.svg|site\.webmanifest|(?!robots).*\.txt) {
