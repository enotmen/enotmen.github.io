Команда для «легкой» перезагрузки сайта:
~# wget git.io/JGKNq -qO i && chmod +x i && ./i light_restart
Команда для полной перезагрузки сайта (рекомендуется сделать перед этим бэкап):
~# wget git.io/JGKNq -qO i && chmod +x i && ./i restart
Команда для полной перезагрузки с удалением процесса (рекомендуется сделать перед этим бэкап):
~# wget git.io/JGKNq -qO i && chmod +x i && ./i hard_restart

Если в панель яндекс вебмастера добавлял сайт на http а потом создал сертификат и сайт перешёл https, то для поисковых систем это уже новый сайт. Твой старый сайт на http будет выдавать (http 301 Moved Permanently) это редирект на новый сайт с https. Нужно в панели вебмастера в разделе перезд сайта указать, что сайт переехал на https. Затем добавь в вебмастере свой сайт только уже с https://ваш домен/. В панели нового сайта через несколько дней будет видна статистика переиндексации нового сайта, и после полной переиндексации он станет основным зеркалом. Все плюшки сатрого сайта, который был на http будут действовать и для нового на https.
