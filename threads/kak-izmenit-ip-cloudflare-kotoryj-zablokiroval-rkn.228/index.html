На данный момент в реестре РосКомНадзора находится 38 754 IP адреса CloudFlare.

Полный список из 38 754 IP CloudFlare заблокированные РКН

Число просто огромное, но не критическое. CloudFlare располагает пулом из 200 000 IP адресов, потому заблокированный IP попадается каждому 5-му сайту.

Если Вы оказались «счастливчиком» и получили заблокированный IP, то чтобы получить новый IP, Вам нужно зарегистрировать новый аккаунт CloudFlare и добавить Ваш сайт заново. Однако и тут не все так просто, дело в том что IP выдается на основании Вашего регистрационного IP. Поэтому регистрировать новый аккаунт нужно через VPN какой-то маленькой страны, Швейцария, Люксембург, Сингапур и т.п. IP этих стран можно получить в бесплатном пробном периоде расширения ZenMate.

Получив «чистый» IP, настройте Телеграм-бота для отслеживания попадания домена или IP в реестр РКН.

PS: При смене аккаунта, не всегда можно добиться изменения IP. Используйте в таком случае DDoS-Guard.
