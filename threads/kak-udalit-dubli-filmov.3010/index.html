Спарсил фильмы этой командой.
# 0 ~ https://api.themoviedb.org/3/movie/popular?api_key=TOKEN&page=[page] ~ results.0.id ~ https://api.themoviedb.org/3/movie/..._to_response=credits,external_ids&language=ru ~ id <> custom.tmdb_id ~ external_ids.imdb_id <> custom.imdb_id ~ "movie" <> type ~ backdrop_path <> pictures ~ poster_path <> poster ~ release_date <> premiere ~ title <> title_ru ~ original_title <> title_en ~ overview <> description ~ vote_average <> rating ~ vote_count <> vote ~ genres.0.name <> genre ~ credits.cast.0.name <> actor <> 5 ~ credits.crew.0.name <> director <> 5 <> job == Director ~ production_countries.0.iso_3166_1 <> country
т.к. на сайте уже были фильмы, помимо новых появились дубли.
cinemapress zero поудаляет фильмы с уникальным описанием.
Как можно удалить дубли? Ведь их очень много.

К тому же имена актёров/режисёров с tmdb спарсились транслитом, а с KP они на русском, и выходит что 2 страницы одного актёра /actor/tom+cruize и /actor/том+круз
Что тоже не правильно. Половина фильмов отображается по одному имени а вторая половина по другому.

Или может запрос другой, который добавил по id кинопоиска, чтоб исключить повторы

cinemapress zero поудаляет фильмы с уникальным описанием.
Нажмите для раскрытия...
Команда не трогает фильмы из админ-панели.
Она удалит все фильмы из автоматической базы.

Если они Вам вновь понадобились, Вы их с лёгкостью можете вернуть, введя NOT, во время подтверждения.
